# Weather-App
Weather App 🌦️
    A modern and responsive Weather App built using HTML, CSS, and JavaScript. This app 
    allows users to search for any city and view real-time weather information using the 
    OpenWeatherMap API.

Features
    Search weather by city name
    Real-time temperature display
    Humidity, wind speed, pressure, and visibility
    Sunrise and sunset timing
    Weather condition icons
    Responsive modern UI design
    Error handling for invalid city names
Technologies Used
    HTML5
    CSS3
    JavaScript (ES6)
    OpenWeatherMap API
How to Use
    Enter a city name
    Click the Get Weather button
    View current weather details instantly
API Used
    OpenWeatherMap API
